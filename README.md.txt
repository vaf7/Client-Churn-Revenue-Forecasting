# Client Churn Risk & Revenue Forecasting for Managed IT Services

## Project Overview

This project predicts client churn risk and forecasts future revenue using machine learning. It helps managed IT service providers identify customers who are likely to leave, understand the reasons for churn, and estimate future business revenue.

---

## Objectives

- Predict clients who are likely to churn in the next 3 months.
- Identify the key factors driving customer churn.
- Forecast revenue for the next 12 months.
- Recommend strategies to reduce churn by at least 5%.

---

## Dataset

A synthetic dataset of **10,000 client records** was created to simulate enterprise client transactions.

The dataset includes:

- Client ID
- Service Type
- Monthly Revenue
- Contract Length
- Support Tickets
- Satisfaction Score
- Payment Delay
- Cloud Usage
- Security Incidents
- Churn Status

Project Constraints Implemented:
- 12% missing financial data
- 18% churn rate (imbalanced dataset)

---

## Project Workflow

1. Generated a synthetic dataset.
2. Introduced 12% missing values.
3. Cleaned the dataset using median imputation.
4. Performed Exploratory Data Analysis (EDA).
5. Built a Random Forest Classifier for churn prediction.
6. Built a Random Forest Regressor for revenue forecasting.
7. Evaluated both machine learning models.
8. Generated business recommendations.

---

## Machine Learning Models

### Churn Prediction
- Random Forest Classifier

Evaluation Metrics:
- Accuracy
- Classification Report
- Confusion Matrix

### Revenue Forecasting
- Random Forest Regressor

Evaluation Metrics:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## Business Insights

- Low customer satisfaction increases churn risk.
- Higher payment delays are associated with customer churn.
- Clients with many support tickets are more likely to churn.
- Longer contract durations improve customer retention.
- Improving customer support and reducing payment delays can help reduce churn.

---

## Recommendations

- Improve customer support response times.
- Increase customer engagement and feedback collection.
- Offer incentives for long-term contracts.
- Monitor high-risk clients proactively.
- Strengthen security services and preventive maintenance.
